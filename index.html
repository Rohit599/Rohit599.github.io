India And Pollution  
A Web Application built in Laravel using MdBootstrap. It provides a quick and smart solution to identify where and what kind of pollution is increasing and report it to the concerned authorities so that immediate action could be taken. It gives the option of sharing the issue through social media like facebook, twitter, and Instagram.
